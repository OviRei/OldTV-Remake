## Download game
If you would like to try out the game while there is no release, you can play the game by doing the following:
1. Clone this branch `git clone --branch Electron https://github.com/OviRei/OldTV-Remake`
1. Open cloned repo
1. Run `npm install`
1. Run `npm start`
